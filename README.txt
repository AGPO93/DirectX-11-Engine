DirectX 11 engine - Andres Pereira agpo93@gmail.com


This engine was developed with the purpose of dipping into the world of computer
graphics, and it features an input system, textures, lighting, a .obj importer, 
instancing, and an A* pathfinding AI units that moves along the grid using squad
mechanics.


********* Instructions **********

Move the camera with WASD and rotate it with the arrow keys.

Press spacebar to start A* pathfinding and movement.

Change the int "goalNode" in ModelClass.h to change the destination grid node.
